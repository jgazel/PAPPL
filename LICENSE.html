.:: Application Project for Students of the Ecole Centrale de Nantes ::.

Description

The aim of this work is to compare the conception of concurrent programming in different languages, to understand how it can be executed and to describe the results we can obtain :
  • A comparison and a critic between the following systems : CTJ, JCSP, CSO.
  • Classical problems of concurrent programming implemented in those languages.
  • A tutorial in CSO for students who want to learn how to use it.
